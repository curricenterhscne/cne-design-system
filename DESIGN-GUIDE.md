# 충남 고교학점제 온마당 — 디자인 시스템 v1.0

> **꾸꾸클럽 디자인 시스템** · C-1 라이트 도미넌트
> 참학력 온라인 공동교육과정 멤버쉽 캐릭터 "꾸꾸클럽" 브랜드 기반
> 최종 갱신: 2026-05

이 문서는 **사람이 읽는 가이드**입니다. 실제 적용은 `tokens/design-tokens.css`와
`components/components.css`를 import 하는 것으로 끝납니다. 토큰·컴포넌트의 변수명과
클래스명이 곧 계약(contract)이며, 이 문서는 그 의미와 사용 원칙을 설명합니다.

---

## 0. 이 시스템이 존재하는 이유

충남 고교학점제 가족 사이트(신규 온마당 + 모든 하위 모듈·웹앱)가 **한눈에 같은 가족으로
인식되도록** 하는 단일 진실원(Single Source of Truth)입니다. 모든 사이트는 이 레포의
CSS를 참조하므로, 색 하나를 바꾸면 전체가 함께 바뀝니다.

대상 사용자는 **고1 학생(메인)·교사·학부모·실무자**이며, 같은 시스템 안에서 학생용 화면과
교사 실무 화면이 자연스럽게 분기합니다.

## 1. 브랜드 정체성 — 꾸꾸클럽

| 요소 | 내용 |
|---|---|
| 이름 | **꾸꾸클럽** ("꾸물(꿈을) 꾸는 클럽") |
| 메시지 | 학교 너머, 학습의 공간을 확장하며 꿈을 꾸는 학생들의 모임. 꾸물꾸물 느리지만 천천히 꿈을 향해 나아가는 우리들. |
| 톤 | 과하게 착하지도 유치하지도 않은데 공감되는 / 학생 친화적 / "노력(갓생)형 학생" |
| 캐릭터 | **토꾸미**(토끼·갓생형) · **팬꾸미**(곰, 무덤덤·성실형, 메인) · **꾸미**(애벌레·서브) |

캐릭터는 공동교육과정 전용이 아니라 **온마당 전체 + 모든 모듈의 공식 마스코트**로 사용합니다.

## 2. 색 — Color

모든 색은 꾸꾸클럽 캐릭터 자료집 PDF에서 직접 추출한 값을 기준으로 합니다.

### 핵심 3색 + 보조

| 역할 | 토큰 | HEX | 용도 |
|---|---|---|---|
| 브랜드 진블루 | `--kkuk-blue-500` | `#4F6EBE` | 헤더·버튼·헤드라인·캐릭터 코 도트 |
| 라이트 블루(배경) | `--kkuk-sky-200` | `#DAF2FC` | **페이지 기본 배경** |
| 그린(액센트) | `--kkuk-green-300` | `#7BCE7C` | 네잎클로버 = "권장/행운/성공" |
| 크림(따뜻한 배경) | `--kkuk-cream-100` | `#FFFFE7` | 자료집/에디토리얼 모먼트 |
| 잉크 | `--kkuk-ink` | `#221F20` | 캐릭터 윤곽선·본문 텍스트 |

각 색은 `50`(가장 옅음) ~ `900`(가장 진함) 램프로 확장되어 있습니다. 진블루는 `800`/`900`이
관리자 화면용 딥 톤으로 준비되어 있습니다.

### 의미 기반 토큰을 우선 사용할 것

컴포넌트를 만들 때는 원시 색(`--kkuk-blue-500`)을 직접 쓰지 말고 **의미 토큰**을 쓰세요.
나중에 다크모드나 톤 조정 시 이 레이어만 바꾸면 전체가 따라옵니다.

```css
/* 권장 */     color: var(--color-brand);
/* 비권장 */   color: var(--kkuk-blue-500);
```

주요 의미 토큰: `--color-bg` `--color-surface` `--color-text` `--color-text-muted`
`--color-brand` `--color-accent` `--color-border` `--color-heading`.

### C-1 도미넌트 원칙

- **기본 배경은 라이트 블루**(`--color-bg` = sky-200). 흰 카드를 그 위에 얹습니다.
- **진블루는 액션·강조·헤더**에 점point으로. 면적을 넓게 쓰지 않습니다.
- **그린은 아껴서**. "권장/행운/성공"의 의미일 때만. 남발하면 신호가 약해집니다.

## 3. 서체 — Typography

- **본문·UI: Pretendard** — 한글 가독성 최우선. (실제 배포 시 자체 호스팅 권장)
- **에디토리얼 모먼트(선택): Noto Serif KR** — 자료집 인용 등에만.
- **데이터·숫자: 등폭 숫자(tnum)** — 187개교 표 정렬용. `.tnum` 또는 `.num` 클래스.

> 캐릭터 워드마크("꾸꾸클럽")의 손글씨체는 **이미지 자산으로만** 사용합니다.
> 웹폰트 손글씨는 본문 가독성을 해치므로 UI 텍스트에 쓰지 않습니다.

### 위계 (5단계 + 보조)

| 토큰 | 크기 | 용도 |
|---|---|---|
| `--text-display` | 39.8px | 페이지 대표 타이틀(히어로) |
| `--text-h1` | 31.3px | 페이지 제목 |
| `--text-h2` | 25px | 섹션 제목 |
| `--text-h3` | 20px | 카드/하위 섹션 제목 |
| `--text-body` | 16px | 본문 (한글 최소 안전 크기) |
| `--text-sm` | 14px | 보조 텍스트·표 셀 |
| `--text-xs` | 12px | 라벨·배지·캡션 |

본문 줄 높이는 `--leading-normal`(1.7)로 한글 가독성을 확보합니다.

## 4. 간격·반경·그림자

- **간격**: 4px 베이스 6단계 — `--space-1`(4px) ~ `--space-10`(64px).
- **반경**: 캐릭터가 둥글둥글하므로 시스템도 부드럽게 — `--radius-sm`(6) `md`(10) `lg`(16) `xl`(24) `pill`.
- **그림자**: 옅고 부드럽게 — `--shadow-sm/md/lg`. 과한 그림자는 캐릭터 톤과 안 맞습니다.

## 5. 핵심 컴포넌트 5 + α

`components/components.css`에 정의. 모든 클래스는 `kk-` 접두사.

| 컴포넌트 | 클래스 | 비고 |
|---|---|---|
| 헤더 | `.kk-header` | 진블루 배경 + 마스코트 + 내비 |
| 버튼 | `.kk-btn` + `--primary/secondary/ghost/accent` + `--sm/lg` | accent=그린 |
| 카드 | `.kk-card` + `--course/hover` | 과목 카드 변형 포함 |
| 입력 폼 | `.kk-field` `.kk-input` `.kk-select` `.kk-searchbar` | |
| 표 | `.kk-table` (`.kk-table-wrap`로 감쌈) | sticky 헤더, 등폭 숫자 |
| 배지 | `.kk-badge` + `--recommend/core/online/neutral/unit` | 의미 매핑 ↓ |
| 빈 상태 | `.kk-empty` | 캐릭터 활용 |
| 섹션 제목 | `.kk-section-title` | PDF의 eyebrow+제목 패턴 |

### 배지 의미 매핑 (중요 — 사이트 간 통일)

| 배지 | 색 | 의미 | course_selector 연동 |
|---|---|---|---|
| `--recommend` ★ | 그린 | 권장 | URL 파라미터 `want` (일반선택+융합선택) |
| `--core` ◆ | 진블루 | 핵심 | URL 파라미터 `core` (진로선택) |
| `--online` | 라이트블루 | 온라인 개설 | 미개설→온라인학교 안내 |

> 기존 두 사이트(course_selector_cne, 2022-curriculum-majors)의 bridge 연동에서
> ★(권장)·◆(핵심) 표시를 이 배지 클래스로 통일하세요.

## 6. 캐릭터 활용 매트릭스

캐릭터 SVG는 `assets/characters/`에 **디자이너 원본을 벡터 그대로 추출**해 제공합니다
(`꾸꾸클럽.ai`에서 추출, 투명 배경). `<img src="assets/characters/paenkkumi.svg">`로 삽입.
색 변경이 필요하면 디자이너에게 변형본을 요청하세요. 자세한 내용은 `assets/README.md` 참조.

### 어디에 누구를

| 위치/상태 | 캐릭터 | 비고 |
|---|---|---|
| 헤더 마스코트 | 팬꾸미 | 메인 캐릭터, 차분해서 상시 노출에 적합 |
| 히어로/환영 | 토꾸미 | 갓생형, 동기부여 |
| 빈 상태(결과 없음) | 토꾸미 | "행운은 준비된 사람의 것" 카피와 함께 |
| 로딩/대기 | 꾸미 + `.kk-empty--loading` | "꾸물꾸물 느리지만" (살짝 흔들리는 모션) |
| 과목 카드 아이콘 | 꾸미 | 작은 장식 |
| 푸터·로고 | `kkukkuk-club-logo.svg` (단체+워드마크) | 가족 정체성 |

> 현재 추출된 자산은 각 캐릭터의 **기본 포즈 + 단체 로고**입니다. 표정 변형(윙크·졸림 등)이나
> 턴어라운드, 활용 포즈가 필요하면 원본 `.ai`에서 추가 추출할 수 있습니다(자료집 6~18쪽).

### 모듈별 캐릭터 배정(제안)

자료집 진로설계 모듈을 만들 때 모듈마다 대표 캐릭터를 두면 일관성이 생깁니다.

- 모듈 1 자기이해 → **토꾸미**(자기 루틴·강점)
- 모듈 2 교육과정 이해 → **팬꾸미**(성실히 편제 이해)
- 모듈 5 활동 설계 → **꾸미**(천천히 나아가는 로드맵)
- 모듈 6 종합 보고서 → **3캐릭터 단체**

### 캐릭터 사용 원칙

- 흰 바디는 어떤 배경색 위에도 올라갑니다(라이트블루·크림·흰색 모두 OK).
- 캐릭터를 **너무 작게(<40px)** 쓰면 디테일이 뭉갭니다. 헤더 마스코트(38px)가 하한.
- 학생 친화 화면에 적극, **교사·행정 화면엔 절제**(헤더 마스코트 정도만).
- 캐릭터 페르소나에 맞는 카피를 함께 쓰면 효과가 큽니다(토꾸미=명언, 팬꾸미=무던함).

## 7. 학생 화면 ↔ 교사·행정 화면 분기

C-1은 라이트 도미넌트가 기본입니다. 187개교 데이터·강사인력풀 등 **행정 화면**은
컨테이너에 `.surface-admin` 클래스를 더하면 같은 색조 안에서 차분하게 분기합니다.

```html
<body class="surface-admin">   <!-- 또는 특정 섹션 컨테이너에 -->
```

`--color-bg`가 차분한 회청색으로, 헤더가 딥블루(`blue-800`)로 전환됩니다.
토큰만 덮어쓰므로 컴포넌트 코드는 그대로입니다.

## 8. 적용 방법 (다른 웹앱에 붙이기)

```html
<!doctype html>
<html lang="ko">
<head>
  <meta charset="utf-8">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css">
  <!-- 디자인 시스템 (CDN 또는 GitHub Pages 경로) -->
  <link rel="stylesheet" href="https://<your-pages>/cne-design-system/tokens/design-tokens.css">
  <link rel="stylesheet" href="https://<your-pages>/cne-design-system/components/components.css">
</head>
<body>
  <!-- 캐릭터를 쓸 경우, 페이지 1회 심볼 정의 후 <use>로 호출 -->
  ...
</body>
</html>
```

기존 사이트에 점진 적용할 때는, 충돌을 피하기 위해 **토큰만 먼저 import**해서
색·서체부터 통일하고, 컴포넌트는 화면 단위로 하나씩 `kk-` 클래스로 교체하세요.
